# 台北市大安區地球科學課程 (2026)

> **地震科學 × 地球物理 × AI/Agent 輔助教學**

[![License: Apache 2.0](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](LICENSE)

## 🎯 專案簡介

本專案是 **2026年台北市大安區地球科學教師研習課程**，由顏達翔老師主講，專為**中小學現職教師**設計。

課程結合 **地震科學、地球物理學、真實資料分析** 與 **現代數位工具**，目標是讓教師能親手完成可帶回學校使用的教材、互動網頁、資料分析活動與地球科學教學作業。

## 📚 核心特色

| 特色 | 內容 |
|------|------|
| 🌍 **地震科學** | 地震波、震級、震央定位、場地效應、建物共振 |
| 📊 **真實資料** | 地震波形、手機感測器、公開測站資料分析 |
| 💻 **數位工具** | GitHub、Google GAS、Vercel、Hugging Face、AI Agent |
| 🎓 **教學轉化** | 將複雜科學概念轉化為中小學生能理解的活動 |

## 🗓️ 課程資訊

- **授課教師**：顏達翔老師
- **課程期間**：2026年7月17日 - 8月21日
- **上課時間**：每週四上午，共6週
- **每次時數**：4小時
- **課程對象**：中小學現職教師
- **課程精神**：教學闊地球科學、互動網頁、資料分析活動、地球科學教學作業

## 🎯 學習目標

1. **建立基礎** - 理解地震科學與地球物理學的核心概念
2. **分析資料** - 使用真實地震資料進行簡單分析
3. **轉化教學** - 將科學概念轉化為中小學活動
4. **熟悉工具** - 操作 GitHub、GAS、Vercel 與 AI Agent
5. **建立流程** - 完成資料收集、自動化與展示流程
6. **每週產出** - 每週完成一項可交付作業
7. **實際測試** - 進行野外實測或簡易地震測試
8. **完成作品** - 完成可帶回學校使用的教學成果

## ⏱️ 每週流程 (4小時)

| 時間 | 活動 |
|------|------|
| 0:00-0:20 | 本週導入 (台灣地震案例、新聞與課程目標) |
| 0:20-1:10 | 核心概念 (地震科學與地球物理基礎) |
| 1:10-2:00 | 資料與案例 (IRIS活動、真實資料或教學案例) |
| 2:00-3:20 | 數位實作 (使用工具製作教材與分析資料) |
| 3:20-4:00 | 分享與回饋 (作業理念、小組分享與作業說明) |
